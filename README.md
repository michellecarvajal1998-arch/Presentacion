# Sistema Gestión Usuarios LINQ

## Descripción

Aplicación desarrollada en C# utilizando una arquitectura de 4 capas, SQL Server y LINQ para la gestión de usuarios.

## Tecnologías Utilizadas

* C#
* Visual Studio
* SQL Server
* LINQ
* GitHub

## Arquitectura de 4 Capas

### Entidades

Contiene las clases que representan los datos del sistema.

### Datos

Gestiona el acceso a la base de datos y las consultas.

### Lógica de Negocio

Implementa las reglas y validaciones del sistema.

### Presentación

Interfaz gráfica desarrollada en Windows Forms.

## Base de Datos

Base de datos: SistemaUsuarios

Tabla Usuarios:

* IdUsuario
* Nombre
* Apellido
* Correo
* Clave
* FechaRegistro

## Funcionalidades

* Registrar usuarios
* Consultar usuarios
* Modificar usuarios
* Eliminar usuarios
